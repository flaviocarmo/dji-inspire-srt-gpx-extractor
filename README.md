## About

Tool for creating GPX from DJI Inspire drone video subtitles.

## Prerequisites

* node.js (with npm)

## Instructions

1. get the '*.srt' file from your drone
2. download dependencies with `npm install`
3. create GPX with `node index.js < subs.srt > out.gpx`
